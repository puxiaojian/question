# question

> 本项目是基于vue2.0实现的一个小玩意，

项目技术架构
vue-cli
vue
vue-resource
scss
webpack

项目地址：（`git clone`）
```shell
git clone https://github.com/puxiaojian/question.git
```
通过`npm`安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))

```
npm install
```
启动服务(http://localhost:8888)

```
npm run dev
```
发布代码

```
npm run build
